## Run Locally

Clone the project

```bash
  git clone https://github.com/Ghanishtkhurana/brief-toes-9591.git
```

Go to the project directory

```bash
  cd brief-toes-9591
```

Install dependencies for frontend as well as backend

```bash
  cd frontend
  npm install
```

```bash
  cd backend
  npm install
```

Start the localhost server

```bash
  cd ../frontend
  npm start
```
